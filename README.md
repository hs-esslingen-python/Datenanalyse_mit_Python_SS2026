### virtuelle Python Umgebung mit _uv_ synchronisieren.

Eingabeaufforderung (Terminal cmd.exe) öffnen und in das Verzeichnis des Repos wechseln dann den folgenden Befehl eingeben:

```cmd
uv sync
```
